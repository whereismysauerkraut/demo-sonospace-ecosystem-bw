Growth Flywheel

Self-reinforcing loop
  1. 1
    More Venues Installed
    Capital deploys to build out studio network — 6 → 12+ spaces. Each new space = new platform node.
    → R-08 (€618K), R-01 (licence fees)
  2. 2
    More Artists Join
    Venue depth attracts more creators. Residencies, touring installations, and commissions expand creator node.
    → R-02 (residencies), R-03 (licensing)
  3. 3
    Content Library Deepens
    300 → 400+ works. Richer library raises the value of every venue licence. Data NE compounds.
    → R-03 (content licensing), IP value ↑
  4. 4
    More Audiences Reached
    Better content + more access points = more experiences. 100K+ listeners. Social NE activates tribal identity.
    → R-04 (presenting €337K '26)
  5. 5
    → Repeat: Stronger Platform
    Audience demand pulls in more artists. Artists pull in more venues. Revenue base diversifies. Each loop raises defensibility.
    → Total: €1.88M '26 · EBITDA 29%
